1. downloan xampp,vscode,git,composer,nodejs
2. run terminal in the folder that you want to install laravel

composer global require laravel/installer   (install laravel installer)
composer create-project --prefer-dist laravel/laravel anyFolderName     (create laravel folder"foldername")

3. run command in new created folder

php artisan serv    (to start sever)

4. open broweser "localhost:8000"(if you want to connect to database)

5. install nodejs into laravel using following command

composer require laravel/ui
php artisan ui vue --auth
php artisan  serv
npm install
npm run dev
